Контрольні запитання та відповіді

  • 1. Що таке CSS і яка його роль у веб-розробці? CSS використовується для стилізації веб-сторінок, надаючи їм кольори, шрифти та розташування.
  • 2. Які основні властивості в CSS і як вони задаються? Основні властивості: color, font-size, margin, padding, background-color. Задаються через селектори.
  • 3. Що таке селектори в CSS? Наведіть приклади різних типів селекторів. Селектори визначають, до яких елементів HTML застосовуються стилі. Приклади: тегові (h1), класові (.example), ідентифікаторні (#id).
  • 4. Яка різниця між класовим та ідентифікаторним селекторами? Класовий селектор (.) використовується для групи елементів, а ідентифікаторний (#) — для унікального елемента.
  • 5. Як записати коментарі в CSS? Коментарі в CSS записуються між /* та */. Наприклад: /* Це коментар */.
  • 6. Як змінити фоновий колір, колір тексту та розмір шрифту для елементу? Використовуються властивості: background-color (фон), color (текст), font-size (розмір шрифту).
  • 7. Як задати межі (borders), відступи (margins) та внутрішні відступи (padding) для елементів в CSS? Межі задаються через border, зовнішні відступи — margin, внутрішні відступи — padding.
